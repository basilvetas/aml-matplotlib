# Setup

If you don't have Pipenv installed:
```bash
brew install pipenv
```
or
```bash
pip install pipenv
```

Install dependencies:
```bash
pipenv install --dev
```

# Testing

Enter virtualenv shell
```bash
pipenv shell
```

Run unit tests
```bash
pytest
```
